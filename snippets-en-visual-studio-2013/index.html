Snippets en Visual Studio 2013

Una de las cosas que he notado cuando desarrollo es que hay ciertas tareas que son muy repetitivas, por ejemplo, normalmente escribo muchas pruebas unitarias, y cada vez que escribo una prueba unitaria sigo un formato el cual es el siguiente:


[Test]
public void MethodUnderTest_Context_Expected()
{
    //Arrange

    //Act

    //Assert
}

Entonces, cada vez que voy a escribir el test, tengo que hacer lo mismo.

Entonces, por que no crear un snippet? Así me evitaría escribir tantas veces lo mismo, y solo tendría que insertarlo y reemplazar valores.

Los snippets son porciones de código predefinidos que podemos insertar facilmente a nuestro código, incluso podemos definir parametros sustituibles y al momento de agregarlo de manera automática podemos reemplazar esos valores.

Snippet template

Para poder crear snippets debemos utilizar un template, el cual está en formado XML y que ha sido definido por Microsoft.

Con el template solo bastaría agregar nuestro código y valores como son:

  • Titulo
  • Shortcut
  • Código a insertar
  • Author, entre otros.

El siguiente es un ejemplo de un snippet template:


<?xml version="1.0" encoding="utf-8"?>
<CodeSnippets
    xmlns="http://schemas.microsoft.com/VisualStudio/2005/CodeSnippet">
    <CodeSnippet Format="1.0.0">
        <Header>
            <Title></Title>
        </Header>
        <Snippet>
            <Code Language="">
                <![CDATA[]]>
            </Code>
        </Snippet>
    </CodeSnippet>
</CodeSnippets>

Create code snippet

Para crear code snippets podemos hacerlo de 2 formas:

1.- Utilizando la herramienta Snippy, la cual es una aplicación Windows forms que nos provee una interfaz bastante simple y la cual nos facilita la creación de snippets.

Pueden bajarla acá.

2.- Creando el archivo XML manualmente utilizando el template.

Para crear el snippet manualmente tenemos que:

  • Creamos un archivo XML en Visual Studio y agregamos el template para snippets.
  • Agregamos un titulo en las etiquetas title.
  • Especificamos el Language, por ejemplo CSharp.
  • Agregamos el código en la sección de CDATA por ejemplo:

<Code Language="CSharp">
    <![CDATA[Console.WriteLine("Hello, World!")]]>
</Code>

Listo ! ya tenemos nuestro snippet, ahora solo queda guardar el snippet con extension .snippet y debemos importarlo a Visual Studio para poder usarlo.

Nota: Ya sea que utilicemos la herramienta Snippy o lo hagamos manualmente, es necesario importarlo a Visual studio.

Agregando Snippet a Visual Studio

  • Vamos a Tools/Code Snippets Manager

  • Damos clic al boton de Import

  • Vamos a la ubicación donde hemos guardado nuestro snippet y damos clic en Open.
  • Después nos preguntará donde queremos agregar el code snippet en las carpetas que aparecen del lado derecho, una de las opciones será My Code Snippets, lo seleccionamos, después Finish y OK.

Y listo ya tenemos el snippet agregado!
