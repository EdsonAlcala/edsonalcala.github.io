Como usar HighlightJS en Ghost blog

Hay ciertas ocasiones que nos gustaría mostrar código en nuestro blog y aunque Ghost por default nos permite agregar código utilizando las etiquetas:

<code> El código que queremos </code>

tal vez no es suficiente para lo que deseamos.

Que tal si podemos mostrar el código como un editor de texto por ejemplo Sublime Text.

Bueno pues es muy fácil, para ello debemos utilizar Highlight.js que es un pluging de JavaScript que nos permite resaltar nuestro código.

Los pasos para realizar esta tarea son las siguientes:

  • Incluir el archivo CSS(hoja de estilos) en el tema que queremos.
  • Incluir la libreria HighlightJS
  • Inicializar la libreria
  • Usarlo

Voy a hacer este pequeño ejemplo utilizando el tema default de Ghost que es Casper, así que nos digirimos a nuestro folder a la ruta ghost/content/themes/casper y abrimos el archivo default.hbs.

Dentro de este archivo buscamos el comentario

{{! Styles'n'Scripts }}

Agregamos el link a la hoja de estilos del tema que deseemos, si queremos otro tema entonces Aquí podemos encontrar más y los enlaces a esos temas estan Aquí


  {{! Styles'n'Scripts }}
  <link rel="stylesheet" type="text/css" href="{{asset "css/screen.css"}}" />
  <link rel="stylesheet" type="text/css" href="//fonts.googleapis.com/css?family=Noto+Serif:400,700,400italic|Open+Sans:700,400" />
    <link rel="stylesheet" type="text/css" href="//cdnjs.cloudflare.com/ajax/libs/highlight.js/8.2/styles/monokai_sublime.min.css">

Lo siguiente será localizar el comentario

{{! The main JavaScript file for Casper }}

agregar el link a la libreria HighlightJS y además la vamos a inicializar.


{{! The main JavaScript file for Casper }}
<script type="text/javascript" src="{{asset "js/jquery.fitvids.js"}}"></script>
<script type="text/javascript" src="{{asset "js/index.js"}}"></script>  
<script src="http://cdnjs.cloudflare.com/ajax/libs/highlight.js/8.2/highlight.min.js"></script>  
<script>hljs.initHighlightingOnLoad();</script>

Para una lista completa de clases consultar Aquí.

Ahora vamos a probarlo, por ejemplo, si queremos mostrar código JavaScript como code snippet, entonces simplemente incluimos nuestro código entre las etiquetas <pre> </pre> y agregamos la clase del lenguage que queremos en la etiqueta <code>.

Por ejemplo para JavaScript: 


    <pre><code class="js">Your JavaScript code here</code></pre>

Ejemplo


$(document).ready(function() {
    console.log('hello');
});

Y para una hoja de estilos CSS: 


    <pre><code class="css">Your css code here</code></pre>

Ejemplo 


code {  
  padding: 2px 4px;
  font-size: 90%;
  color: #c7254e;
  background-color: #f9f2f4;
  border-radius: 4px;
}

Finalmente, para desactivar el pluging usamos la clase nohighlight.

<pre><code class="nohighlight">This text is not highlighted.</pre></code>

Y listo, ya podemos usar HighLightJS en nuestro blog para resaltar nuestro código y que se vea bonito.
