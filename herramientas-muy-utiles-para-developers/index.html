Herramientas muy útiles para developers

SQL DBX

http://www.sqldbx.com/

Nimble Text

http://nimbletext.com/

Cuando descubri Nimble Text en un post de Scott Hanselman, nunca imagine el gran poder que estaba al alcance de mis manos, permite manipular texto y tiene muchas funciones, más tarde agregaré un post acerca de que podemos hacer con esta poderosa herramienta, pero definitivamente, tienes que tenerla en tu repertorio.

Fiddler

http://www.telerik.com/fiddler

Herramienta para inspeccionar el tráfico desde y hacia tu computadora. Algo muy padre por que incluso permite ver los headers, y toda la petición HTTP.

Atom

https://atom.io/

Es un editor de texto para markdown, es muy padre, por el momento no puedo escribir más al respecto por que lo acabo de bajar, pero lo usa el equipo de Azure de Microsoft y lo recomiendan ampliamente, de hecho si quieres ser contribuidor tienes que usar este editor. Más adelante contaré mi experiencia, pero a primera vista, se ve muy padre.

Postman Rest Client

https://chrome.google.com/webstore/detail/postman-rest-client/fdmmgilgnpjigdojojpjoooidkmcomcm

Excelente herramienta para realizar pruebas y llamadas de Rest APIs.

Json Viewer

http://jsonviewer.stack.hu/

Permite inspeccionar si los datos de entrada estan en formato JSON, y además permite inspeccionar el objeto JSON.

XML Viewer

http://codebeautify.org/xmlviewer

Excelente herramienta en linea para poner inspeccionar elementos con formato XML.
